How To Use:

The local student manager is a tool designed to assist teachers keep track of student participation. This version was designed to serve in tandem with the official version on seattleacademy.courses. It will save your class list locally, but do not rely on it to save your lists long term.
To add students to a class, use the text box below the save and load buttons. You must manually enter your student's names in the box following this format: Firstname Lastname, Firstname Lastname, Firstname Lastname... Once you've finished, press 'create list' to generate the students.
Using the (+) button next to a students name will allow you to keep track of how many times they have participated. Useful information will be displayed at the top, such as which student has participated the most and what the average engagement (based on this metric) is. You can use the 'clear all student points' button to clear it. You can also use the 'pick student with zero points' button to get a random student that has no participated.
If you'd like to save your students, select a block using the drop down menu, and press the save button. Now, if you press the load button with that block selected your saved students will reappear. Saving again will overwrite the old student list for each block.
